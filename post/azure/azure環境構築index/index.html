Azure 環境構築 index

January 14, 2021
  • リソースグループを追加する
    • サブスクリプションにリソースグループを追加し仮想マシンやストレージなどのリソースを追加出来るようにします。
  • 仮想ネットワークを追加する
    • Azure 上でネットワークを構築します。
  • ストレージアカウントを追加する
    • ローカルと Azure 上の仮想マシンでファイルをやり取りするためのストレージを用意します。
  • 仮想マシンを追加する
    • Azure 上に仮想マシンを構築します。
  • SQL 仮想マシンを追加する
    • Azure 上に SQL Server を載せた仮想マシンを構築します。
  • Bastion を追加する
    • Web ブラウザで Azure のポータルから Azure の仮想マシンに接続出来るようにします。
  • Azure Files を追加する
    • Azure の仮想マシンでストレージをマウントしローカルとファイルをやり取り出来るようにします。
  • SSMS を日本語化する
    • Azure で SQL Server を含む仮想マシンを構築すると英語版の SSMS がプリインストールされてしまうため日本語化したい場合は追加で設定が必要です。
  • クォータ上限を引き上げる
    • Azure ではコア数の上限をクォータといい性能評価でマシンサイズを上げるときにクォータ上限の引き上げをサポート経由で依頼する必要があります。
  • Authenticator 追加
    • Azure のアカウントに Microsoft Authenticator で二要素認証出来るようにします。
