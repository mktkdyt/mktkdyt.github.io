Azure Files 設定手順

January 14, 2021

概要

Azure Files は Azure でファイル共有を可能にするサービスです。
仮想マシンにストレージアカウントをマウントさせることが出来ます。

手順

Azure のポータル画面で [ストレージアカウント] をクリックします。

ファイル共有したい [ストレージアカウント] をクリックします。

[File サービス]/[ファイル共有] をクリックします。

[＋ファイル共有] をクリックします。

下記内容を入力し [作成] をクリックします。

項目 設定
名前 share
クォータ 20
層 トランザクションが最適化されました

作成したファイル共有をクリックします。

[接続] をクリックします。

PowerShell コマンドをコピーします。

仮想マシンに接続し PowerShell コマンドを実行します。

再起動しても自動的にマウントされるようになります。
